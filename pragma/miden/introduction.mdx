## What is Pragma Miden?

Pragma Miden brings Pragma's oracle infrastructure to [Polygon Miden](https://docs.polygon.technology/miden/), a zero-knowledge virtual machine that enables client-side proving and strong privacy guarantees.

Unlike traditional oracle models, Pragma Miden leverages the Miden account model: publishers write prices directly to their own on-chain accounts, and the oracle aggregates across all registered publishers at query time — entirely on-chain, with no off-chain intermediary.

## Architecture

### How it works

<Steps>
  <Step title="Publishers push prices">
    Each publisher submits a self-transaction that writes a price entry (pair, price, decimals, timestamp) into their own account storage.
  </Step>
  <Step title="Oracle aggregates on query">
    When a consumer queries `median`, the Oracle account fetches the latest entry from every registered publisher and computes the median — on-chain, verifiably.
  </Step>
  <Step title="Result returned">
    The oracle returns `is_tracked`, `median_price`, and an optional `amount` parameter — all verifiable by the Miden proof system.
  </Step>
</Steps>

|-----------|---------|
| `1:0`     | BTC/USD |
| `2:0`     | ETH/USD |

| `4:0`     | BNB/USD |
| `5:0`     | XRP/USD |
| `6:0`     | HYPE/USD |
| `7:0`     | POL/USD |
