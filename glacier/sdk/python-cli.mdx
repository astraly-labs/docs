## Installation

```bash
pip install pragma-glacier
```

The `glacier` CLI is installed alongside the Python SDK.

## Authentication

### Login

```bash
glacier auth login
```

Prompts for your API key and stores it in `~/.pragma/config.toml`.

### Check Status

```bash
glacier auth status
```

Shows whether you are authenticated and your key prefix.

### Environment Variable

```bash
export GLACIER_API_KEY="glc_your_key"
```

### `glacier files`

List available files for a dataset and date range.

```bash
glacier files --dataset orderbooks --from 2026-03-01 --to 2026-03-15
```

| Flag | Required | Description |
|------|----------|-------------|
| `--dataset` | Yes | Dataset topic |
| `--from` | Yes | Start date (YYYY-MM-DD) |
| `--to` | Yes | End date (YYYY-MM-DD) |
| `--source` | No | Filter by source |
| `--pair` | No | Filter by pair |

Example output:

### `glacier download`

Download files to a local directory.

```bash
glacier download --dataset orderbooks --from 2026-03-01 --to 2026-03-15 -o ./data
```

| Flag | Required | Description |
|------|----------|-------------|
| `--dataset` | Yes | Dataset topic |
| `--from` | Yes | Start date |
| `--to` | Yes | End date |
| `-o`, `--output` | No | Output directory (default: `./data`) |
| `--source` | No | Filter by source |
| `--pair` | No | Filter by pair |
| `--parallel` | No | Number of parallel downloads (default: 4) |

Example:

```bash
$ glacier download --dataset trades --from 2026-03-01 --to 2026-03-07 -o ./data --parallel 8

Done! 84 files saved to ./data
```
