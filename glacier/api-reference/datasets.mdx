---
title: "List Datasets"
description: "Returns all available datasets with their schemas."
---

## Request

<CodeGroup>
```bash cURL
curl -H "Authorization: Bearer glc_your_key" \
  https://data.pragma.build/v1/datasets
```
```python Python
from pragma_glacier import GlacierClient

client = GlacierClient()
datasets = client.list_datasets()
for ds in datasets:
    print(f"{ds.name}: {ds.description}")
```
</CodeGroup>

No query parameters required.
